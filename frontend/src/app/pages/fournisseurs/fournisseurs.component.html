🍪 Récupérer la session de votre navigateur

But : donner au robot une page où le magasin est déjà choisi, pour qu'il voie les prix que le site vous cache autrement. On copie pour cela les cookies et le localStorage de votre navigateur, une fois le magasin sélectionné.

Étape 1 — Préparer la page

  1. Ouvrez le site du fournisseur dans votre navigateur (Chrome, Edge, Firefox…).
  2. Choisissez votre magasin / drive / mode de livraison, exactement comme pour commander.
  3. Vérifiez que les prix des produits s'affichent bien : c'est le signe que la session est prête.

Étape 2 — Exporter la session

  1. Ouvrez la console : touche F12 (ou clic droit → « Inspecter »), puis onglet Console.
  2. Copiez le code ci-dessous, collez-le dans la console, appuyez sur Entrée.
{{ snippetSession }}

Ce code lit uniquement les cookies et le stockage du site affiché et les copie dans votre presse-papier. Il n'envoie rien nulle part. Si la console affiche « Allow pasting » (Chrome/Edge la première fois), tapez ces deux mots, validez, puis recollez le code.

Étape 3 — Coller dans la fiche fournisseur

  1. Le résultat est déjà dans votre presse-papier (le code se termine par copy(...)).
  2. Revenez ici, dans le champ Session du navigateur, et collez (Ctrl+V).
  3. Enregistrez le fournisseur, puis lancez 🤖 Synchroniser.

💡 Le robot rafraîchit cette session à chaque synchronisation réussie : vous n'aurez à refaire cette manipulation que si le magasin change, ou si les prix cessent d'apparaître (session expirée). Dans ce cas, videz-la avec le bouton 🍪✕.
